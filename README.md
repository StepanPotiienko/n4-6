# КП №5: Дослідження сигналу Н4-6 (методичний алгоритм)

Цей Django-застосунок реалізує варіант 5 для КП:
дослідження короткочасової нестабільності вихідного сигналу калібратора Н4-6
у вигляді розширеної невизначеності `U0.95(t)`.

## Що реалізовано

1. Зчитування 120 відліків з `data/raw_signal.txt` (формат `1,049990700E+00`).
2. Перетворення в масив відносних відхилень у ppm:
   `delta_i = (x_i - mean_x) / mean_x * 1e6`.
3. Ручний розрахунок статистик (без `numpy`/`statistics`):
   - середнє,
   - стандартна помилка,
   - медіана,
   - мода,
   - стандартне відхилення,
   - дисперсія,
   - ексцес,
   - асиметричність,
   - інтервал,
   - мінімум,
   - максимум,
   - сума,
   - відлік.
4. Сортування масивів виконується алгоритмом `Quick Sort`.
5. Для кожного вікна `1, 2, 5, 10, 30, 60` хв:
   - формуються поточні інтервали,
   - будуються гістограми за формулою Стерджеса `n = 1 + 3.322 * lg(N)`,
   - обчислюються `n_j`, `B_j`, `H_N`,
   - визначаються межі `U0.95`.
6. Формуються:
   - Таблиця 1 (статистики),
   - Таблиця 3 (поточні інтервали),
   - Таблиця 4 (осереднені `U0.95(t)`).
7. Поліном 3-го порядку (МНК) для `U_low(t)` і `U_up(t)`.
8. Інтерфейс, CSV-експорт і друк.

## Структура

- `analysis/services.py` — усі розрахунки за методичним алгоритмом.
- `analysis/views.py` — формування контексту та CSV-експорт.
- `analysis/templates/analysis/index.html` — таблиці, графіки, гістограми.
- `analysis/tests.py` — тести для парсингу, quick sort, статистик і структури таблиць.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
