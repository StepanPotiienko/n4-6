Вихідний сигнал x(t)

Графік первинних значень сигналу. Дозволяє візуально оцінити загальну поведінку, наявність дрейфу, шумів та грубих помилок (викидів).

Відносне відхилення δ(t), ppm

Сигнал, центрований відносно середнього значення та виражений у ppm. Відображає чисту нестабільність незалежно від номіналу сигналу.

Загальна гістограма відхилень (HN_global)

Візуалізує статистичний розподіл усього сигналу. Форма розподілу свідчить про характер процесів у джерелі сигналу та наявність детермінованих завад чи дрейфу.

U0.95(t) та поліном 3-го порядку

Ключовий графік аналізу. Точки показують осереднені значення довірчих меж для різних вікон. Лінія — апроксимація цих значень поліномом, що використовується для математичного моделювання нестабільності.

Гістограма вікна (абсолютна)

Розподіл значень у межах першого сегменту обраного вікна. Показує частоту появи значень у конкретних діапазонах відхилень.

Гістограма вікна (нормована HN)

Нормована за частотою гістограма вікна. Пунктирна лінія — рівень 1−P = {{ crossing_level }} (P = {{ controls.coverage }}). Перший і останній стовпчик вище цього рівня — межі U{{ controls.coverage }}.
